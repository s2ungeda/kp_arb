# 다른 PC 설치 (배포)

새 PC에서 kp-arb를 돌리기 위한 절차. 운영 진입점은 **main.bat 하나**다.

## 1. 준비물

- **Python 3.11 이상** — <https://www.python.org/downloads/> 에서 설치.
  설치 화면에서 **"Add python.exe to PATH" 체크 필수**.
- 프로젝트 폴더 — 둘 중 하나:
  - `git clone https://github.com/s2ungeda/kp_arb.git`
  - 또는 기존 PC의 폴더를 통째로 복사 (`.venv` 폴더는 복사하지 말 것 — PC마다 새로 만든다)

## 2. 설치

프로젝트 폴더에서 **`setup.bat`** 더블클릭 — 가상환경(.venv)을 만들고 의존성을 설치한다.

## 3. 비밀값(API 키) 옮기기

키는 git에 **없다**(비밀). 둘 중 한 방법:

- **간단**: 기존 PC의 `.env` 파일을 프로젝트 폴더로 복사.
- **권장(자격증명관리자)**: 새 PC에서 하나씩 등록 —

  ```text
  .venv\Scripts\python.exe -m kp_arb.secrets_cli set LS_STOCK_APPKEY
  (LS_STOCK_APPSECRET / LS_STOCK_ACCT / LS_STOCK_ACCT_PW,
   LS_DERIV_... 4종, HL_AGENT_KEY, HL_ACCOUNT_ADDRESS 동일)
  ```

- 실행 모드: 기본은 모의(paper). 운영이면 `.env`에 `KP_MODE=live`.

## 4. 실행

**`main.bat`** 더블클릭 → 메뉴 ▸ 코어 ▸ 코어 시작 → 화면 메뉴에서 필요한 화면 열기.
마지막 상태(코어·화면·입력값)는 자동 저장되어 다음 실행 때 복원된다.

## 5. 주의

- 같은 키로 **두 PC에서 동시에 접속하지 말 것** — LS 세션이 충돌할 수 있다.
- 시세 모니터는 현재 자체 접속(과도기)이라 모니터만 띄워도 LS/HL에 연결된다.
- 문제가 생기면 코어 콘솔 창의 메시지와 `logs/` 폴더를 확인.
